# Web Art Shorts Bot

Automated YouTube Shorts — har video me ek **unique interactive web page** (canvas animation).
Har baar different physics, objects, colors, patterns. Kabhi repeat nahi.

## How it works

1. `generator.py` — 15+ unique templates, har baar random combination
2. `record.py` — Playwright se page record karta hai
3. `upload.py` — YouTube pe upload (OAuth)
4. GitHub Actions — daily 5 shorts auto

## Templates

cloth, particles, wave, pendulum, gravity, tree, fire, metaball,
flowfield, orbit, spiral, noise, cellular, bubble, maze, rain, aurora, galaxy
